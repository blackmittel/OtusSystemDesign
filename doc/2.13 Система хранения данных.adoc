=== Система хранения данных

*PostgreSQL* — основная реляционная СУБД для транзакционных данных

* БД Пользователей: данные пользователей, учетные записи, роли, сессии, токены доступа.
* БД Объявлений: товары, услуги, категории, цены, остатки, состояние (активно/снято).
* БД Заказов: заказы, позиции в заказе, история статусов (создан → оплачен → доставлен), связь с оплатой и доставкой.

*Elasticsearch* — поисковый движок и хранилище логов

* Кластер поиска: индексы объявлений с текстовыми и атрибутными данными для быстрого поиска и фильтрации.
* Кластер логов: централизованное хранение логов со всех сервисов и инфраструктуры.

*Redis* — кеш и хранилище сессий (in-memory)

* сессии пользователей, JWT-токены, кеш популярных объявлений, кеш результатов поисковых запросов

*S3-совместимое объектное хранилище* — хранение файлов

* изображения товаров, аватары пользователей, документы (сканеры доверенностей, спецификации). Erasure coding для отказоустойчивости.

*RabbitMQ* — брокер сообщений (хранилище сообщений)

* события бизнес-процессов (создание объявления, новый заказ, изменение статуса оплаты/доставки). Сообщения хранятся до обработки, зеркалируются на 2 узла.

*ClickHouse* — хранилище для аналитики и отчетности

* денормализованные данные заказов, платежей, пользовательских действий и метрик сервисов.
