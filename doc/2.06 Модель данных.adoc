=== Модель данных

User — пользователь системы (покупатель или сотрудник платформы).

Role — роль пользователя (покупатель, продавец, администратор, модератор, аналитик).

UserRole — связка пользователя и его ролей.

SellerProfile — профиль продавца с дополнительными данными (рейтинг, тип, реквизиты).

Ticket — объявление о продаже товара или услуги.

Moderation — статус и результат модерации объявления.

Conversation — диалог между пользователями (набор сообщений), по объявлению.

Message — отдельное сообщение в диалоге.

Order — заказ, связывает покупателя, продавца и объявление.

Payment — операция оплаты заказа покупателем.

Payout — операция выплаты средств продавцу.

Transaction — финансовая проводка (движение денег внутри системы).

Shipment — доставка заказа с трекингом.

DeliveryService — служба доставки (внешний провайдер).

Category — категория (рубрика) объявлений.

FavoriteList — список избранных объявлений пользователя.

Notification — уведомление пользователю о событиях системы.

Review — отзыв пользователя о другом пользователе или сделке.
