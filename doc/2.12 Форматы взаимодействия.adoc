=== Форматы взаимодействия

==== HTTPS / REST (JSON)
основной формат всех внешних и межсервисных API-вызовов

* Клиенты -> API Gateway
* API Gateway -> все бизнес-сервисы
* Сервис заказов -> Сервис платежей (инициация оплаты)
* Сервис заказов -> Сервис доставки (создание доставки)
* Сервис объявлений -> Сервис поиска (прямое обновление индекса)
* Сервис поиска -> Elasticsearch
* Интеграции с платежными провайдерами
* Интеграции со службами доставки

==== AMQP
асинхронная коммуникация через брокер сообщений

* Все бизнес-сервисы -> RabbitMQ (публикация событий)
* RabbitMQ -> Сервис поиска (обновление индекса)
* RabbitMQ -> Аналитический сервис (сбор статистики)
* RabbitMQ -> Redis (инвалидация кэша)

==== SQL
взаимодействие с реляционными базами данных

* Сервис авторизации -> БД Пользователи (PostgreSQL)
* Сервис объявлений -> БД Объявления (PostgreSQL)
* Сервис заказов -> БД Заказы (PostgreSQL)
* Аналитический сервис -> реплики БД (только чтение)

==== Redis
работа с кэшем

* Сервис авторизации -> Redis (сессии, токены)
* Сервис объявлений -> Redis (кеш объявлений)
* Сервис поиска -> Redis (кеш результатов)

==== HTTP / REST (S3 API)
работа с объектным хранилищем

* Сервис объявлений -> S3-хранилище (загрузка/получение изображений)

==== протоколы обслуживания
логи, мониторинг, трассировка
