==== HTTP/REST (JSON)
Основной формат синхронного взаимодействия.

[cols="15,30,30", options="header"]
|===
| Компонент A | Компонент B | Описание

| Client
| API Gateway
| Вызовы бизнес-сервисов (пользователи, объявления, поиск, заказы, платежи)

| API Gateway
| Сервис пользователей
| Управление профилем и списками

| API Gateway
| Сервис объявлений
| Создание/редактирование объявлений, сообщения

| API Gateway
| Сервис поиска
| Поиск и рекомендации

| API Gateway
| Сервис заказов
| Создание/обновление заказа

| API Gateway
| Платёжный сервис
| Инициация платежа

| Сервис расчётов
| Платёжный сервис
| Выплаты продавцам
|===

==== HTTP - медиа передача

[cols="15,30,30", options="header"]
|===

| Client

|===

==== gRPC (Protobuf)
Формат внутренних синхронных внутри ЦОД.

[cols="15,30,30", options="header"]
|===
| Компонент A | Компонент B | Описание

| API Gateway
| Сервис авторизации
| Проверка токена / аутентификация пользователя

| Сервис заказов
| Платёжный сервис
| Создание платежа в рамках заказа

| Сервис заказов
| Сервис доставки
| Расчёт стоимости и параметров доставки

| Сервис расчётов
| Платёжный сервис
| Выплаты и учёт финансов
|===
