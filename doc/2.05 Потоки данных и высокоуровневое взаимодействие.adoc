=== Потоки данных и высокоуровневое взаимодействие

[cols="15,20,30,15,30", options="header"]
|===
| ID | Поток данных | Участники | Тип | Описание

| IF-1
| Резолвинг и маршрутизация
| Клиент → GeoDNS → API Gateway
| Синхронный
| Геораспределение трафика и маршрутизация клиента в ближайший ЦОД (Москва/Краснодар)

| IF-2
| Получение медиа-контента
| Клиент → CDN
| Синхронный
| Доставка статического и медиа-контента через CDN (СПб, Казань)

| IF-3
| Аутентификация и авторизация
| API Gateway → Сервис авторизации
| Синхронный
| Проверка токена и аутентификация пользователя для всех защищённых операций

| IF-4
| Управление пользователем
| Клиент → API Gateway → Сервис пользователей
| Синхронный
| Получение и изменение профиля, управление списками пользователя

| IF-5
| Работа с объявлениями
| Клиент → API Gateway → Сервис объявлений → Сервис медиа
| Синхронный
| Создание, редактирование объявлений, модерация, обмен сообщениями, загрузка/получение медиа

| IF-6
| Поиск и рекомендации
| Клиент → API Gateway → Сервис поиска
| Синхронный
| Выполнение поисковых запросов и получение рекомендаций

| IF-7
| Управление заказом
| Клиент → API Gateway → Сервис заказов → Сервис доставки
| Синхронный
| Создание заказа, расчёт стоимости и параметров доставки

| IF-8
| Инициация платежа
| Клиент → API Gateway → Платёжный сервис
| Синхронный
| Инициация платежной транзакции пользователем

| IF-9
| Создание платежа из заказа
| Сервис заказов → Платёжный сервис
| Синхронный
| Создание платежа в рамках оформления заказа

| IF-10
| Выплаты продавцам
| Сервис расчётов → Платёжный сервис
| Синхронный
| Выполнение выплат и финансовых операций в пользу продавцов

| IF-11
| Кэширование данных
| API Gateway → Кэш (Redis)
| Синхронный
| Кэширование часто используемых данных (сессии, объявления, результаты поиска)

| IF-12
| Обработка результата платежа
| Платёжный сервис → Kafka → Сервис заказов / Сервис расчётов
| Асинхронный
| Передача результата платежа для обновления статуса заказа и учёта средств

| IF-13
| Жизненный цикл заказа
| Сервис заказов → Kafka → Сервис доставки
| Асинхронный
| Публикация события создания заказа и инициирование доставки

| IF-14
| Обновление статуса доставки
| Сервис доставки → Kafka → Сервис заказов / Аналитический сервис
| Асинхронный
| Передача статусов доставки и обновление состояния заказа и аналитики

| IF-15
| Учёт и движение средств
| Сервис расчётов → Kafka → Аналитический сервис
| Асинхронный
| Передача информации о балансе, транзакциях и выплатах

| IF-16
| Индексация и рекомендации
| Сервис объявлений → Kafka → Сервис поиска
| Асинхронный
| Передача изменений объявлений для индексации и обучения рекомендаций

| IF-17
| Поведенческие события
| Сервис поиска → Kafka → Аналитический сервис
| Асинхронный
| Сбор пользовательского поведения для аналитики и улучшения рекомендаций

| IF-18
| Сообщения пользователей
| Сервис объявлений → Kafka → Аналитический сервис
| Асинхронный
| Передача событий сообщений между пользователями для аналитики и уведомлений

| IF-19
| Уведомления и аналитика
| Kafka → Аналитический сервис
| Асинхронный
| Централизованная обработка событий для аналитики и генерации уведомлений
|===
