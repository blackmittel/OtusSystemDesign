=== Схема взаимодействия

==== Принципы взаимодействия

===== Синхронные взаимодействия
Используются когда:
- требуется мгновенный ответ пользователю
- нужна строгая консистентность
- операции короткие

Протоколы:
- HTTPS (REST)
- gRPC

===== Асинхронные взаимодействия
Используются когда:
- нет требования мгновенного ответа
- требуется устойчивость к сбоям
- есть несколько подписчиков
- не требуется атомарная целостность

Протокол:
- Kafka
